# Hammerspoon

## Why?

Because I can't get Karabiner-Elements working on this computer without IT's
help, and they haven't responded to my request.

## Installation

Downloaded the zip file for the latest release on
https://github.com/Hammerspoon/hammerspoon/releases, unzipped it, and dragged
the app into `/Applications`.

Granted control access in System Settings:

- _Privacy & Security_
  - _Accessibility_
    - Enable Hammerspoon
